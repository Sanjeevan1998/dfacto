# Dfacto

**Status**: Work in Progress (WIP)

This is a monorepo for Dfacto, an application for real-time fact-checking and an agentic crawler. 

## Project Structure

The repository is divided into two main components:

- **`backend/`**: A FastAPI application using LangGraph for web crawling and data extraction. It integrates with various search APIs (such as Tavily, Reddit, and NewsAPI) to gather data and stores results in a SQLite database.
- **`frontend/`**: A Flutter cross-platform mobile/web application designed to display the fact-checking and crawling results.
